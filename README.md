# 프로젝트 개요
- 디자인에 중점을 둔 코딩사이트
- 기존 코딩 사이트들의 지루한 디자인이 코딩사이트에 대한 거부감을 유발.
- 유니크한 디자인의 코딩사이트를 제작하고 코딩사이트에 대한 거부감을 줄이고 싶음.
